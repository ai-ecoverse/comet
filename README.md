# Comet - Experience Intelligence Skill

A SLICC skill for analyzing websites and generating visual HTML reports. No backend, no deployment - the AI agent is the analyst, generator, and chat interface.

## Features

- **Website Analysis** - Crawl sites, fetch sitemaps, query indexes, analyze metadata
- **Visual Reports** - Generate self-contained HTML reports with charts, stats, and insights
- **Design System** - Includes a complete CSS framework for consistent, beautiful reports
- **Report Hub** - Centralized navigation across all generated reports

## Usage

Install as a skill in SLICC, then ask:

- "analyze {url}"
- "audit {url}"
- "generate a report on {url}"
- "check the SEO/brand/performance of {url}"
